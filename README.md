# THE UNREMEMBERED · 未被记得者

> **A Weird-Fiction Mystery in Ten Chapters · 十章怪谈**
> 灰谷镇的人不会失踪。他们只是不再被记得。

一部**单文件、零依赖**的沉浸式互动叙事游戏。打开 `index.html` 即可游玩（推荐戴耳机、Chrome/Edge/Safari 桌面或移动端）。

```
▶ 直接双击 index.html
▶ 或本地起服:  npx http-server .   →  http://localhost:8080
▶ 或开启 GitHub Pages 指向仓库根目录
```

---

## 这是什么

以小镇超自然悬疑的情感语法为骨架、为 Z 世代重写的原创 IP：这一代的怪物更安静——倦怠、现实解体、拟社会哀伤、在线上表演自我的压力、以及"活着却被遗忘"的现代恐惧。

核心设计原则是 **机制即隐喻（ludonarrative resonance）**：每一个系统都是让玩家"演出来"而非"读出来"的修辞。

## 系统 → 玩家会亲手碰到的实现

| 设计文档系统 | 在游戏里的样子 |
|---|---|
| **清醒值 Lucidity** | 莫依的感知完整度。潜暮会消耗它；低于阈值时**界面本身开始撒谎**——环境文本被改写、背包出现复影格、频率表显示不存在的章节标题 |
| **稳定值 Stability** | 帕里斯的情绪承重。压抑中起草的结构带有**只有玩家看得见**的暗裂（第二/三/六章的三处抉择），全部在第八章按入账顺序逐一兑付 |
| **共鸣 Resonance** | 二人组状态。第四章争吵时**机制上整段不可用**（音乐同时被切断），关系修复因此成为玩法需求而非过场 |
| **未寄出 Undelivereds** | 既是钥匙、又是文献、又是道德货币：送达治愈薄暮，攥住则让「寂」更厚。第二章你会在「道歉」专柜遇到一件署名是你自己的 |
| **燃忆 Memory-burning** | 第五章叫醒空心人需要烧掉第 1–4 章真实获得的记忆——**存档真的跟着遗忘**：档案条目变虚线、地图记忆线熄灭、第七章最软的对话选项消失、第九章的"真实锚点"少一个 |
| **噤 Hush** | 第七章解锁的第四声区。所有冲突中的最优解、零消耗——但每次使用，**整个游戏画面会永久去掉一格饱和度**。游戏全程对你是否相信它自己的论点进行公投 |
| **不可靠界面** | 第一章冷开场：血条、小地图、暂停键、对话框里的名字被逐一夺走。第八章稳定值仪表被从屏幕上移除；清醒值归零后第九章整章菜单撒谎、目标自动完成、三个莫依里只有一个与你**实际玩过的版本**重量吻合 |
| **FEEL IT** | 终章最后一个交互：按住按钮听完整首闭幕歌——包括难听的那一段。松手歌会停在原地等你；它不数你松手的次数。不按完，没有职员表 |

章节结构为 **branch-and-bottleneck**：每章结尾固定，但"伤痕"（谁被叫醒、哪些记忆幸存、暗裂几道、对森娜说了什么、是否用过噤）全部持久化并重新着色后续所有章节与三种以上结局组合。

## 视觉与声音

- **Analog dusk**：钠灯琥珀 × 磁带青 × CRT 辉光（扫描线、孔栅、色差、噪点、暗角），Canvas 实时绘制灰谷镇黄昏——屋脊、电线、闪着红灯的广播塔
- **薄暮世界 soft-wrong**：色相偏移、呼吸的边缘（`breathe` 动画）、倒悬于天空的镇子剪影、上升的讯息之雪、琥珀色记忆线
- **全程序生成音频**（WebAudio，无任何音频文件）：氛围持续音、电台空气底噪、四种声区音色；摇篮曲动机以纯净/小调/跑调/完整四种变体在 3、4、9、10 章回归——**跑调版属于第九章的帕里斯**
- 静默是被设计的事件：第四章争吵时音乐被硬切

## 可访问性 / 其他

- 设置中可关闭噪点/闪烁/故障效果（光敏感）、三档文字速度、音量
- 进度自动存档于每章结束（localStorage）；二人组 HUD、档案、地图全部与存档实时同步
- 全部内容在一个 `index.html` 内，无构建、无网络依赖（字体为渐进增强）
